## Quick Start

1. **For Ollama (Local)**: Download from [ollama.com](https://ollama.com), run `ollama run llama2`
2. **For Cloud AI**: Get API key from [ai.google.dev](https://ai.google.dev) (free!)
3. Install mod from Geode
4. Open editor, click AI button
5. Click lock icon, enter key (skip for Ollama)
6. Type prompt, optionally enable AI naming/descriptions
7. Generate!

## Models

For longer and more complicated levels, it is **Highly** recommended you use the best model possible.

## How It Works

1. You describe a level
2. AI generates JSON with object positions
3. Mod creates objects in editor
4. You refine and save!
