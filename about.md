# Editor AI

Generate Geometry Dash levels using AI! Supports Gemini, Claude, and ChatGPT with direct API integration.

## Features

- **3 AI Providers** - Gemini (free), Claude, ChatGPT
- **9 Models** - Choose speed vs quality
- **Direct Integration** - No server needed
- **Secure Keys** - Stored locally
- **Easy Setup** - Get key, click lock icon, generate
- **Customizable** - Difficulty, style, length, max objects

## Quick Start

1. Get API key from [ai.google.dev](https://ai.google.dev) (free!)
2. Install mod from Geode
3. Open editor, click AI button
4. Click lock icon, enter key
5. Type prompt, generate!

## Models

**Gemini (FREE):**
- gemini-1.5-flash (fast)
- gemini-1.5-pro (quality)
- gemini-2.0-flash-exp

**Claude (Paid):**
- claude-3-5-sonnet (balanced)
- claude-3-5-haiku (fast)
- claude-3-opus (best)

**ChatGPT (Paid):**
- gpt-4o (latest)
- gpt-4o-mini (cheap)
- gpt-4-turbo (balanced)

## Settings

- **AI Provider** - gemini/claude/openai
- **Model** - 9 options
- **Difficulty** - easy/medium/hard/extreme
- **Style** - modern/retro/minimalist/decorated
- **Length** - short to xxl
- **Max Objects** - up to 1,000,000

## How It Works

1. You describe a level
2. AI generates JSON with object positions
3. Mod creates objects in editor
4. You refine and save!

## Privacy

- Keys stored locally only
- No telemetry
- Direct API calls to your chosen provider

## Requirements

- Geometry Dash 2.2074
- Geode mod loader
- API key (Gemini is free!)
- Internet connection

## Tips

- Start with Gemini (free)
- Be specific in prompts
- Save before generating
- Try different models
